
  # Book de Produto

  This is a code bundle for Book de Produto. The original project is available at https://www.figma.com/design/G33VjK6tVYSzbCLeFtNwbB/Book-de-Produto.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Local browser preview

  This project is a Vite/React app, so the root `index.html` will not work when opened directly as a `file://` URL.

  To preview without a dev server, run `npm run build` and open `dist/index.html` in your browser.
